3.Usage
*********************************************************************
the applcation requires a login from the user using email and a
password, 
Some preloaded users you can use include 
an employee user, email:employee1@gmail.com password:@Password1
and  a farmer user, email:johndoe@gmail.com password:@Password1
The application prompts the Employee user for input into textboxes in the 
windows. The application asks the user to enter Employee email,
Password, Password confirmation ,and role via dropdown list when 
entering a a new user. a confirm account will then apear and the employee 
will click the provided confirm link.

Employee can search for a user farmer using a farmers email address,
if the Employee wishes to filter the results by thw date of entry the 
empoyee can enter a start and end date for a search.

The application prompts the farmer user for input into textboxes in the 
windows. The application asks the user to enter Stock Name,
Stock descripion, Quantity ,entry date (via calendar),
and stock type via dropdown list when entering a stock item. 
The User then can select diffrent functions to update study time,start
a new week of study and view a graph by clicking on the buttons to do 
so for the specific module on the view modules page.
